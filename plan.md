# PinSnap Archive 기획 요약

## 핵심 문제

사용자는 인스타그램, 릴스, 블로그에서 가고 싶은 장소를 발견하지만 저장된 링크와 실제 방문 계획이 분리되어 있습니다. 이 프로젝트는 발견한 장소를 지도 기반의 개인 아카이브로 바꾸고, 동선과 일정까지 이어주는 것을 목표로 합니다.

## MVP

1. 캡처/공유 텍스트 분석
   - 해시태그, 지명, 장소명을 기반으로 장소 후보를 추론합니다.
   - 실제 배포에서는 OCR, SNS 공유 Intent, Places API 검증을 연결합니다.

2. 카테고리별 지도 저장
   - 카페, 맛집, 숙박, 관광지로 분류합니다.
   - 지도에는 카테고리별 마커와 상세 카드가 표시됩니다.

3. 구역 묶기와 동선 정렬
   - 사용자가 지도 위 구역을 선택하면 주변 저장 장소를 추출합니다.
   - 가까운 순서 기반 휴리스틱으로 방문 순서를 제안합니다.

4. 코스 공유와 일정표
   - 추천 동선을 공유 가능한 링크 패키지로 만듭니다.
   - 방문 시간과 장소별 팁을 일정표로 보여줍니다.
